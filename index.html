Selected Work

TruthMindr — Multimodal AI Agent

Personal project · 2025

Prototype agent that processes text, images, and metadata using an end-to-end flow: OCR “Perception” → NLI “Consistency” → metadata “Evidence” → final “Arbiter” decision with a trace.

Disinformation Detection with CLIP, ViLT & FLAVA

Master’s thesis · ACM DHOW’25

Multimodal disinformation detection on Fakeddit using vision–language models and metadata fusion. Includes extensive evaluation, ablations, and error analysis.

GCP ELT Pipeline & Knowledge Graph

Data & analytics project

ELT pipeline in BigQuery with Bronze/Silver/Gold layers, schema validation, CI/CD checks and Looker dashboards, built to feed a downstream knowledge-graph and exploration UI.
